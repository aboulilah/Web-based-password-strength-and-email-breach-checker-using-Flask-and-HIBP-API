A Web-Based System for Evaluating Password Strength and Email Breach Risk Using Entropy and Verified Breach Intelligence.

## Features
- Real-time password strength evaluation using entropy and pattern analysis
- Detection of weak patterns such as repeated characters
- Email breach checking using Have I Been Pwned API
- Password breach detection using k-anonymity hashing
- OWASP secure coding practices implemented
- Flask-based backend with HTML, CSS, and JavaScript frontend

## Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- Have I Been Pwned API
- zxcvbn password strength library
